# Notas Less #


### Importar archivos ###

Para importar archivos debemos hacerlo con la clausula **@import**, seguido se la ruta del archivo, luego el nombre del archivo y por ultimo la extensión de dicho archivo.

*Ejemplo:* <br>

`@import "base/_tipografia.less";`
